# Twitch Emotes Portfolio

A custom portfolio site showcasing Twitch emotes created for Fiverr and clients.

## About

This is a single-page HTML portfolio designed to display custom Twitch emote work. Perfect for showcasing designs to potential clients and on Fiverr.

## Features

- Clean, responsive design
- Easy to navigate portfolio showcase
- Client work demonstrations

## Usage

Simply open `index.html` in your web browser to view the portfolio.
